# USAF PFRA Calculator 2026

A modern, mobile-friendly U.S. Air Force Physical Fitness Readiness Assessment (PFRA) Calculator designed to help current and future Airmen estimate their fitness assessment score based on the latest Air Force fitness standards.

🌐 Live Website: https://iamraj-patel.github.io/afpt/

---

## ✈️ Features

✅ Calculate projected PFRA scores

✅ Support for multiple fitness components

- 2-Mile Run
- 20-Meter HAMR
- Push-Ups
- Hand-Release Push-Ups
- Sit-Ups
- Cross-Leg Reverse Crunches
- Forearm Plank
- Waist-to-Height Ratio (WHtR)

✅ Age Group Selection

✅ Male and Female Scoring Tracks

✅ Real-Time Score Updates

✅ Responsive Design

- Desktop
- Laptop
- Tablet
- iPad
- Mobile Devices

✅ Modern Air Force-Themed User Interface

✅ GitHub Pages Hosted

## 🎯 Purpose

This project was created to help future Airmen, active-duty members, and fitness-minded individuals quickly estimate their Air Force Physical Fitness Readiness Assessment score while preparing for official testing.

The goal is to provide a simple, accessible, and mobile-friendly calculator that can be used anywhere without requiring account creation or software installation.
