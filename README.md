# USAF PFRA Calculator 2026

Responsive GitHub Pages calculator for estimating U.S. Air Force PFRA scores using the official PFRA scoring charts provided in this project workflow.

## Included events

- 2-mile run
- 20-meter HAMR
- 1-minute push-ups
- 2-minute hand-release push-ups
- 1-minute sit-ups
- 2-minute cross-leg reverse crunches
- Forearm plank
- Waist-to-height ratio body composition

## Notes

- Uses the uploaded Air Force logo image as the website logo.
- All score tables are embedded locally in `standards.js`.
- Logo and favicon assets are bundled locally.
- No cookies, localStorage, tracking, or saved progress are used.
- Website created and maintained by Raj Patel.
- This is an unofficial planning tool. Verify official record scores with current AFPC/DAFMAN guidance and your UFPM/FAC.
